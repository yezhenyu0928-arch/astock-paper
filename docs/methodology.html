因子与风险模型

因子体系总览

本项目参考 MSCI Barra 中国A股模型（CNE5/CNE6）与 Axioma Robust Risk Model， 按免费数据现实裁剪，实现 10 个风格因子。每个因子由 1-3 个描述符加权复合。 当前 S1 v3（红利7因子）和 S4 v2（多因子7因子）各使用其中 7 个因子。

因子描述符S1 v3S4 v2说明
Size（市值）ln(总市值)✅ 正向✅ 负向S1偏大盘防御，S4偏小盘弹性
Beta（贝塔）60日滚动超额收益对市场回归斜率✅ 负向✅ 正向S1偏好低Beta防御，S4牛市好高弹性
Momentum（动量）RSTR+6月+3月—✅ 正向12-1月动量+多窗口复合
Value（价值）1/PE+1/PB+DY✅ 正向✅ 正向三描述符合成，红利策略核心因子
Volatility（波动）DASTD+ATR✅ 负向✅ 过滤S1低波加分，S4剔除高残差波动股
Quality（质量）ROE-杠杆代理✅ 正向✅ 正向高ROE+低杠杆=盈利可持续

Earnings Yield（盈利收益）1/PE(TTM)✅ 正向✅ 正向与VALUE互补衡量估值

数据处理管线

  1. 去极值（MAD Winsorize）：对每个因子截面，用中位数绝对偏差（MAD）设定上下界， 越界值截断到边界。公式：bound = median(x) +/- 5 x 1.4826 x MAD(x)。防止个别的极端数值扭曲整体评估。
  2. 标准化（Z-score）：z = (x - mu_w) / sigma_eq，其中 mu_w 为市值加权均值，sigma_eq 为等权标准差。 处理后，因子分布以市值加权组合为中心(暴露约=0)，等权标准差约=1。
  3. 正交化（Gram-Schmidt）：按固定顺序(BETA→SIZE→VALUE→...→EARNINGS_YIELD)依次对前一因子做WLS回归取残差。 消除因子间的共线性——例如残差波动与Beta天然相关，正交化后残差波动不再包含Beta已解释的部分。
  4. 缺失处理：缺失的描述符在复合时按可得权重重归一，标准化后NaN填0（池中性）。

风险模型（Barra 横截面法）

结构模型：r = Xf + u（个股收益 = 因子暴露 x 因子收益 + 特异收益）。

  1. 暴露矩阵 X：N只股票 x 6个风险因子(size, beta, momentum, resvol, liquidity, btop)的当天暴露值。
  2. 因子收益估计：对每个交易日，用t-1日暴露对t日个股收益做WLS（权=sqrt(市值)）横截面回归，得因子收益 f_t。
  3. 因子协方差 F：f_t 的EWMA协方差（半衰期90日），x252年化。
  4. 特异波动 σ_i：残差 u_i 的EWMA标准差（半衰期42日），xsqrt(252)年化。
  5. 组合预测波动：σ_p = sqrt( h'X F X'h + Σ h_i^2 σ_i^2 )，h=各持仓市值权重。
  6. 组合暴露：X_p = Σ h_i · z_i。因标准化以市值加权均值为中心，X_p 本身即为主动暴露（相对于市值加权基准）。

对称性说明（暴露怎么看）

暴露值为正 → 组合在该因子上比市值加权基准偏多（如正Beta = 比市场Beta更高）。
暴露值为负 → 组合在该因子上比基准偏少。
暴露值在[-0.5, 0.5] → 基本中性，无明显偏离。
暴露值>|1| → 显著偏离，需要注意该维度的集中风险。
