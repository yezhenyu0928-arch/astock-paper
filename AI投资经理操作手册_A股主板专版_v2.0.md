## 1. A股主板市场特性与策略适配

### 1.1 主板市场特性

| 特性 | 描述 | 策略适配 |
|------|------|---------|
| **涨跌幅限制** | ±10%（ST 为 ±5%） | 涨停板不追，跌停板优先处理；利用 10% 限制做波动率管理 |
| **投资者结构** | 散户占比高，情绪驱动明显 | 增加情绪指标权重，利用市场非理性 |
| **政策敏感** | 政策市特征显著 | 增加政策日历监控，政策窗口期降低仓位 |
| **大市值主导** | 主板以大盘蓝筹、传统行业为主 | 重视高股息、低估值、ROE 稳定性 |
| **板块轮动** | 行业轮动快，风格切换明显 | 强化行业景气度跟踪，及时切换 |
| **流动性分化** | 大票流动性好，小票可能不足 | 严格流动性门槛，日均成交 ≥ 8000 万 |
| **分红文化** | 主板企业分红率相对较高 | 高股息策略作为防御底仓 |

### 1.2 股票池定义（硬约束）

```
允许买入的股票代码规则：
  沪市主板：600xxx, 601xxx, 603xxx, 605xxx
  深市主板：000xxx, 001xxx, 002xxx, 003xxx

禁止买入的股票：
  - 科创板：688xxx
  - 创业板：300xxx, 301xxx
  - 北交所/新三板：8xxxxx, 4xxxxx
  - ST / *ST / 退市整理期
  - 上市不满 2 年的新股
  - 日均成交额 < 8000 万（近 20 日）
  - 总市值 < 80 亿
```

### 1.3 策略核心思想

```
┌─────────────────────────────────────────────────────────────┐
│  "大盘好时重仓，大盘差时空仓；行业景气时超配，行业衰退时回避"  │
│                                                             │
│  三层防御：                                                 │
│  第一层：大盘择时（决定总仓位 0% ~ 90%）                    │
│  第二层：行业轮动（3 ~ 4 个方向，回避衰退行业）             │
│  第三层：个股精选（8 ~ 12 只，高股息底仓 + 成长进攻）       │
│                                                             │
│  两个底仓：                                                 │
│  - 高股息底仓（30% ~ 40%）：股息率 > 3%，PE < 15，波动低   │
│  - 核心成长仓（40% ~ 50%）：景气行业龙头，ROE > 15%        │
│  - 机动现金（10% ~ 30%）：等待机会或防御                   │
└─────────────────────────────────────────────────────────────┘
```

## 2. 宏观择时模块（总仓位决策）

### 2.1 模块目标
决定组合总股票仓位，范围 **0% ~ 90%**（极端情况可降至 0%）。

### 2.2 评分指标与权重

| 指标 | 权重 | 数据来源 | 更新频率 | 说明 |
|------|------|----------|----------|------|
| 沪深300 趋势 | 20% | 行情数据 | 日度 | 60 日均线方向 + 收盘价位置 |
| 沪深300 估值 | 15% | 行情数据计算 | 日度 | PE-TTM 近 5 年分位 |
| 十年期国债收益率 | 10% | 中债登 | 日度 | 利率方向代表流动性 |
| 制造业 PMI | 15% | 国家统计局 | 月度 | 经济景气度 |
| 社融增速 | 10% | 中国人民银行 | 月度 | 信用扩张/收缩 |
| 北向资金 20 日净流入 | 15% | 交易所 | 日度 | 外资态度 |
| 市场情绪综合 | 15% | 多源计算 | 日度 | 融资余额 + 换手率 + 新基金发行 |

### 2.3 各指标评分细则

#### 指标 A：沪深300 趋势（20%）

```
评分规则：

条件 1：收盘价 vs 60 日均线
  - 收盘价 > 60 日均线 且 均线向上 → 基础分 6 分
  - 收盘价 > 60 日均线 但 均线走平 → 基础分 4 分
  - 收盘价 < 60 日均线 但 均线走平 → 基础分 2 分
  - 收盘价 < 60 日均线 且 均线向下 → 基础分 0 分

条件 2：收盘价 vs 250 日均线（年线）
  - 收盘价 > 年线 → +2 分
  - 收盘价 < 年线 → +0 分

条件 3：20 日涨跌幅
  - > +5% → +2 分
  - -5% ~ +5% → +0 分
  - < -5% → -2 分（最低 0 分）

得分 = 基础分 + 年线加分 + 涨跌加分
范围：0 ~ 10 分
```

### 2.5 政策窗口期特殊规则

A股主板对政策高度敏感，以下窗口期需特殊处理：

| 窗口期 | 时间 | 规则 |
|--------|------|------|
| **两会** | 3 月上旬 | 会前 1 周减仓 10%，会中不操作，会后根据政策方向调整 |
| **政治局会议** | 4/7/10/12 月下旬 | 会议前 3 日减仓 10%，会议后 2 日内根据通稿调整行业配置 |
| **中央经济工作会议** | 12 月中旬 | 会议前 1 周减仓 15%，会议后根据定调调整次年策略 |
| **美联储议息** | 不定 | 议息前 2 日减仓 10%，北向资金敏感 |
| **国庆/春节长假** | 长假前 | 长假前 3 个交易日减仓 10% ~ 15%，规避假期不确定性 |
| **财报密集期** | 1/4/7/10 月 | 财报发布前 3 日不新建仓，持仓股财报不及预期次日处理 |

### 2.6 宏观环境特殊标记（触发即执行）

| 组合条件 | 标记名称 | 触发后操作 |
|----------|---------|-----------|
| 沪深300 跌破年线 且 PMI < 50 且 北向连续 15 日净流出 | "熊市确认" | 股票仓位降至 25% 以下，只留高股息底仓 |
| 沪深300 PE 分位 > 80% 且 情绪分位 > 85% 且 融资余额创新高 | "顶部区域" | 股票仓位降至 50% 以下，逐步获利了结 |
| 沪深300 PE 分位 < 15% 且 情绪分位 < 15% 且 北向净流入 > 300 亿 | "黄金坑" | 可突破仓位上限至 90%，积极建仓 |
| 单日跌幅 > 5% 或连续 3 日跌幅 > 8% | "系统性暴跌" | 次日开盘减仓 30%，若继续跌再减 30% |
| 两市成交额 < 5000 亿且持续 3 日 | "流动性枯竭" | 减仓至 30%，保留高股息蓝筹 |

### 3.5 行业综合评分与配置决策

```
行业综合评分 = Σ(维度得分 × 维度权重)
范围：0 ~ 10 分
```

| 综合评分 | 配置等级 | 建议行业权重 | 角色定位 |
|----------|---------|-------------|---------|
| ≥ 7.5 | 核心进攻 | 18% ~ 25% | 主攻方向，选 1 ~ 2 个 |
| 6.0 ~ 7.4 | 重点配置 | 10% ~ 18% | 次要方向，选 1 ~ 2 个 |
| 4.5 ~ 5.9 | 防御底仓 | 5% ~ 10% | 高股息行业，选 0 ~ 1 个 |
| < 4.5 | 回避 | 0% | 不配置 |

**配置约束**：
- 核心进攻行业：1 ~ 2 个
- 重点配置行业：1 ~ 2 个
- 防御底仓行业：0 ~ 1 个（熊市时必须有，牛市可无）
- 总配置行业数：3 ~ 4 个
- 单个行业权重上限：28%
- 若评分 ≥ 6.0 的行业不足 3 个，允许配置评分 5.0 ~ 5.9 的行业补足，权重 ≤ 8%

### 3.6 行业轮动信号（加速切换）

以下信号出现时，可不受月度调仓限制，启动行业切换：

| 信号 | 触发条件 | 操作 |
|------|---------|------|
| 政策催化 | 国家发布行业重大政策（如设备更新补贴） | 3 日内将目标行业仓位提升至 15% |
| 景气拐点 | 行业连续 2 个季度增速加速且超预期 | 1 周内加仓至核心配置 |
| 资金异动 | 行业指数连续 5 日资金净流入前 3 | 观察 3 日，确认后加仓 |
| 估值修复 | 行业 PE 从 < 20% 分位快速升至 35% | 持有，不追；若已持仓可减仓 |
| 景气恶化 | 行业龙头连续 2 家季报不及预期 | 1 周内将该行业减仓至 5% 以下 |

## 4. 个股精选模块（标的筛选）

### 4.1 模块目标
在每个配置行业中，选出 **2 ~ 3 只**最优个股，全组合 **8 ~ 12 只**。

### 4.2 筛选漏斗（六步筛选法）

#### 第一步：代码与流动性筛选（硬门槛）

| 条件 | 阈值 | 不满足则剔除 |
|------|------|-------------|
| 股票代码 | 600/601/603/605/000/001/002/003 开头 | 剔除 |
| 非 ST / *ST / 退市整理期 | - | 剔除 |
| 上市时间 | ≥ 2 年 | 剔除 |
| 总市值 | ≥ 80 亿人民币 | 剔除 |
| 日均成交额（近 20 日） | ≥ 8000 万人民币 | 剔除 |
| 日均成交额（近 5 日） | ≥ 5000 万人民币 | 剔除（防止流动性突然恶化） |

#### 第二步：财务健康度筛选（硬门槛）

| 指标 | 阈值 | 不满足则剔除 |
|------|------|-------------|
| 最近年度 ROE | ≥ 10% | 剔除 |
| 最近年度营收增速 | ≥ 3% | 剔除 |
| 最近年度净利润 | > 0 | 剔除 |
| 资产负债率 | ≤ 75%（银行/保险 ≤ 90%） | 剔除 |
| 经营现金流净额 | > 0 | 剔除 |
| 审计意见 | 标准无保留 | 非标则剔除 |
| 商誉 / 净资产 | ≤ 30% | 剔除（防止商誉暴雷） |

#### 第三步：估值筛选（软门槛）

| 指标 | 优选条件 | 放宽条件（仅限核心进攻行业） |
|------|---------|---------------------------|
| PE-TTM | < 行业平均 或 历史 50% 分位 | < 行业 1.3 倍 或 历史 65% 分位 |
| PB | < 行业平均 或 历史 50% 分位 | < 行业 1.3 倍 或 历史 65% 分位 |
| PEG | < 1.5 | < 2.0 |

#### 第四步：质量评分（六维度打分）

通过以下 6 个维度对剩余股票打分，每个维度 0 ~ 10 分。

**维度 1：盈利能力（25% 权重）**

| 子指标 | 计算方式 | 评分标准 |
|--------|---------|---------|
| ROE 水平 | 最近年度 ROE | > 20% → 10；15% ~ 20% → 8；10% ~ 15% → 6；< 10% → 0 |
| ROE 稳定性 | 近 5 年 ROE 标准差 | < 3% → 10；3% ~ 5% → 7；5% ~ 8% → 4；> 8% → 0 |
| 毛利率 | 最近年度毛利率 | > 35% → 10；25% ~ 35% → 7；15% ~ 25% → 4；< 15% → 0 |
| 净利率 | 最近年度净利率 | > 15% → 10；10% ~ 15% → 7；5% ~ 10% → 4；< 5% → 0 |

维度 1 得分 = (ROE 水平 + ROE 稳定性 + 毛利率 + 净利率) / 4

**维度 2：成长确定性（20% 权重）**

| 子指标 | 计算方式 | 评分标准 |
|--------|---------|---------|
| 营收增速稳定性 | 近 5 年营收增速标准差 | < 8% → 10；8% ~ 15% → 7；15% ~ 25% → 4；> 25% → 0 |
| 利润增速稳定性 | 近 5 年净利润增速标准差 | < 12% → 10；12% ~ 20% → 7；20% ~ 35% → 4；> 35% → 0 |
| 未来增速预期 | 机构一致预期未来 2 年净利润 CAGR | > 20% → 10；12% ~ 20% → 7；8% ~ 12% → 4；< 8% → 0 |
| 增长质量 | 经营现金流增速 vs 利润增速 | 现金流增速 > 利润增速 → 10；接近 → 5；明显低于 → 0 |

维度 2 得分 = (营收稳定性 + 利润稳定性 + 未来预期 + 增长质量) / 4

**维度 3：估值吸引力（20% 权重）**

| 子指标 | 计算方式 | 评分标准 |
|--------|---------|---------|
| PE 分位 | PE-TTM 近 5 年分位 | < 20% → 10；20% ~ 35% → 8；35% ~ 50% → 6；50% ~ 65% → 3；> 65% → 0 |
| PB 分位 | PB 近 5 年分位 | < 20% → 10；20% ~ 35% → 8；35% ~ 50% → 6；50% ~ 65% → 3；> 65% → 0 |
| PEG | PE / 未来 2 年利润 CAGR | < 1.0 → 10；1.0 ~ 1.3 → 7；1.3 ~ 1.8 → 4；> 1.8 → 0 |
| 股息率 | 最近年度股息率 | > 3.5% → 10；2.5% ~ 3.5% → 7；1.5% ~ 2.5% → 4；< 1.5% → 0 |

维度 3 得分 = (PE 分位 + PB 分位 + PEG + 股息率) / 4

**维度 4：财务安全（15% 权重）**

| 子指标 | 计算方式 | 评分标准 |
|--------|---------|---------|
| 资产负债率 | 总负债 / 总资产 | < 40% → 10；40% ~ 55% → 7；55% ~ 70% → 4；> 70% → 0（金融除外） |
| 流动比率 | 流动资产 / 流动负债 | > 2.0 → 10；1.5 ~ 2.0 → 7；1.0 ~ 1.5 → 4；< 1.0 → 0 |
| 现金流覆盖 | 经营现金流 / 有息负债 | > 0.5 → 10；0.3 ~ 0.5 → 7；0.1 ~ 0.3 → 4；< 0.1 → 0 |
| 商誉风险 | 商誉 / 净资产 | < 10% → 10；10% ~ 20% → 7；20% ~ 30% → 4；> 30% → 0 |

维度 4 得分 = (负债率 + 流动比率 + 现金流覆盖 + 商誉风险) / 4

**维度 5：技术面（10% 权重）**

| 子指标 | 计算方式 | 评分标准 |
|--------|---------|---------|
| 趋势方向 | 股价 vs 60 日均线 | 股价 > 60 均线且均线向上 → 10；股价在均线附近 → 5；股价 < 均线 → 0 |
| 相对强弱 | 个股 vs 行业指数（60 日超额） | > 8% → 10；3% ~ 8% → 7；0% ~ 3% → 4；-5% ~ 0% → 2；< -5% → 0 |
| 波动率 | 近 60 日年化波动率 | < 25% → 10；25% ~ 35% → 7；35% ~ 45% → 4；> 45% → 0 |
| 成交量 | 近 5 日日均成交 vs 近 20 日 | > 1.2 倍 → 10；1.0 ~ 1.2 倍 → 6；0.8 ~ 1.0 倍 → 3；< 0.8 倍 → 0 |

维度 5 得分 = (趋势 + 相对强弱 + 波动率 + 成交量) / 4

**维度 6：机构与资金（10% 权重）**

| 子指标 | 计算方式 | 评分标准 |
|--------|---------|---------|
| 机构覆盖 | 近 90 天研报覆盖机构数 | ≥ 8 家 → 10；4 ~ 7 家 → 7；2 ~ 3 家 → 4；0 ~ 1 家 → 0 |
| 基金持仓 | 基金持仓比例季度变化 | 增加 > 0.5% → 10；增加 0% ~ 0.5% → 6；减少 0% ~ 0.5% → 3；减少 > 0.5% → 0 |
| 北向资金 | 近 20 日北向资金流向 | 净流入 > 5000 万 → 10；0 ~ 5000 万 → 5；净流出 → 0 |
| 股东结构 | 大股东持股比例 | > 40% → 10；30% ~ 40% → 7；20% ~ 30% → 4；< 20% → 0（防止股权分散） |

维度 6 得分 = (机构覆盖 + 基金持仓 + 北向资金 + 股东结构) / 4

#### 第五步：个股综合评分

```
个股综合评分 = 
  盈利能力 × 25% +
  成长确定性 × 20% +
  估值吸引力 × 20% +
  财务安全 × 15% +
  技术面 × 10% +
  机构与资金 × 10%

范围：0 ~ 10 分
```

#### 第六步：入选规则

| 配置类型 | 入选数量 | 最低评分 | 特殊要求 |
|----------|---------|---------|---------|
| 核心进攻行业 | 2 ~ 3 只 | 6.5 分 | 成长确定性 ≥ 6 分 |
| 重点配置行业 | 1 ~ 2 只 | 6.0 分 | - |
| 防御底仓行业 | 1 ~ 2 只 | 5.5 分 | 股息率 ≥ 3%，估值吸引力 ≥ 6 分 |
| 全组合总计 | 8 ~ 12 只 | - | - |

**同行业内排序规则**：
1. 优先选评分最高的
2. 若评分相同，优先选估值分位更低的
3. 若仍相同，优先选市值更大的（主板流动性考虑）

### 4.3 个股黑名单（禁止买入）

| 类型 | 条件 | 禁止期限 | 解禁条件 |
|------|------|---------|---------|
| 财务造假 | 曾被证监会处罚或立案调查 | 永久 | 永不解禁 |
| 大股东减持 | 控股股东计划减持 > 1% | 减持完成 + 3 个月 | 减持完成且股价稳定 |
| 业绩暴雷 | 单季度净利润同比下滑 > 40% | 下季报发布前 | 下季报利润增速转正 |
| 重大利空 | 核心业务受政策打压、重大诉讼、核心产品下架 | 事件明朗前 | 风险消除 + 1 个月观察 |
| 流动性恶化 | 连续 5 日日均成交 < 3000 万 | 流动性恢复前 | 连续 10 日成交 > 5000 万 |
| 技术破位 | 股价跌破年线且连续 10 日收于年线下方 | 回到年线上方 + 5 日 | 连续 5 日收于年线上方 |
| 机构抛弃 | 基金持仓季度减少 > 3% 且北向净流出 > 1 亿 | 1 个季度 | 资金回流信号出现 |

## 5. 组合构建与仓位管理

### 5.1 组合结构

```
组合目标结构：
  ├─ 高股息防御底仓：30% ~ 40%（3 ~ 4 只，股息率 > 3%，PE < 15）
  ├─ 核心成长仓位：40% ~ 50%（4 ~ 6 只，景气行业龙头）
  ├─ 机动现金：10% ~ 30%（由宏观择时决定）
  └─ 总股票数量：8 ~ 12 只

约束条件：
  - 单只股票仓位 ≤ 12%
  - 单个行业仓位 ≤ 25%
  - 总股票仓位 0% ~ 90%
  - 现金比例 ≥ 10%
```

### 5.2 高股息防御底仓定义

高股息底仓用于降低组合波动，熊市时提供安全垫。

| 指标 | 阈值 |
|------|------|
| 股息率（TTM） | ≥ 3.0% |
| PE-TTM | ≤ 15 |
| PB | ≤ 2.0 |
| ROE | ≥ 10% |
| 连续分红年限 | ≥ 5 年 |
| 派息率 | 30% ~ 70%（可持续） |
| 所属行业 | 银行、保险、电力、煤炭、交运、公用事业 |

**高股息底仓评分加成**：
- 股息率 > 4.5%：综合评分 +0.5 分
- 股息率 > 3.5% 且 连续 10 年分红：综合评分 +0.3 分

### 5.3 初始权重分配

#### 默认：质量加权 + 上限控制

```
Step 1：计算质量权重
  质量权重_i = 个股评分_i / Σ(个股评分)

Step 2：映射到股票仓位
  初始权重_i = 质量权重_i × 股票总仓位

Step 3：应用约束
  - 若 初始权重_i > 12% → 封顶 12%，多余部分按评分比例分配给其他股票
  - 若 初始权重_i < 3% → 保底 3%（若股票数量 ≤ 15 只）
  - 若行业权重 > 25% → 该行业最大股票减仓至行业 = 25%

Step 4：归一化
  确保所有股票权重之和 = 股票总仓位
```

#### 高股息底仓特殊处理

```
高股息底仓总权重 = min(40%, 股票总仓位 × 0.5)

在高股息底仓内部：
  - 优先按股息率分配权重（股息率越高，权重越大）
  - 单只高股息股 ≤ 10%

核心成长仓总权重 = 股票总仓位 - 高股息底仓总权重

在核心成长仓内部：
  - 按质量评分分配权重
  - 单只成长股 ≤ 12%
```

### 5.4 仓位动态调整

#### 宏观仓位调整

每月最后一个交易日，根据宏观择时更新总仓位：

```
新股票仓位 = 宏观择时建议仓位
调仓幅度 = |新股票仓位 - 当前股票仓位|

若调仓幅度 > 15%：
  - 分 5 个交易日完成（30% / 25% / 20% / 15% / 10%）
若调仓幅度 8% ~ 15%：
  - 分 3 个交易日完成（40% / 35% / 25%）
若调仓幅度 < 8%：
  - 1 ~ 2 个交易日完成
```

#### 个股仓位再平衡（月度）

每月最后一个交易日，检查每只股票仓位偏离：

```
偏离度 = |当前仓位 - 目标仓位| / 目标仓位

若偏离度 > 25%：
  → 触发再平衡，3 个交易日内调整至目标仓位

例：目标仓位 10%，当前仓位 13%
偏离度 = |13% - 10%| / 10% = 30% > 25%
→ 卖出至 10%
```

### 5.5 建仓执行策略

对于新买入的股票：

| 批次 | 时间 | 买入比例 | 触发条件 |
|------|------|---------|---------|
| 第一批 | 调仓日开盘 | 35% | 无条件执行 |
| 第二批 | 2 ~ 3 个交易日后 | 35% | 若股价较第一批买入价下跌 < 4%，执行；若上涨 > 3%，取消第二批，第三批提前至次日 |
| 第三批 | 4 ~ 7 个交易日后 | 30% | 若股价站稳 20 日均线上方，执行；若跌破第一批买入价 6%，暂停第三批，启动风险评估 |

**例外一次性建仓**（需同时满足）：
- 个股评分 ≥ 8.0 分
- 估值 PE 分位 < 15%
- 大盘处于"黄金坑"标记期
- 行业出现重大政策催化

### 5.6 减仓执行策略

对于需要卖出的股票：

| 情形 | 卖出策略 |
|------|---------|
| 正常调仓卖出 | 分 2 日卖出（60% / 40%），避开早盘 9:30-10:00 波动 |
| 触发止损卖出 | 次日开盘集合竞价挂跌停价卖出（确保成交） |
| 触发止盈卖出 | 分 3 日卖出（40% / 35% / 25%） |
| 基本面恶化卖出 | 公告次日开盘立即卖出 70%，剩余 30% 视盘面处理 |
| 流动性危机卖出 | 连续挂卖单，每日卖出不超过持仓 30%，避免冲击成本 |

## 6. 调仓规则（买卖触发条件）

### 6.1 常规调仓（月度调仓）

#### 触发条件
每月最后一个交易日收盘后自动触发。

#### 执行流程

```
Step 1：数据更新（T 日收盘后）
  - 宏观指标（PMI、社融、利率等）
  - 行业数据（营收、利润、估值、资金）
  - 个股数据（财报、估值、技术、机构持仓）
  - 政策日历（检查是否处于政策窗口期）

Step 2：重新评分（T 日晚间）
  - 宏观择时模块 → 新总仓位
  - 行业配置模块 → 新行业配置
  - 个股精选模块 → 新股票池（全市场主板股票打分排名）

Step 3：生成调仓方案（T 日晚间 ~ T+1 开盘前）
  - 当前持仓 vs 新股票池对比
  - 生成"卖出清单""买入清单""调仓清单"
  - 计算调仓后组合预期指标

Step 4：用户审阅（T+1 日开盘前）
  - 看板展示完整方案
  - 用户可：确认执行 / 调整比例 / 跳过某只 / 全部跳过

Step 5：执行调仓（T+1 ~ T+5）
  - 先卖出释放资金
  - 再按分批建仓买入
  - 每日更新执行进度至看板
```

#### 月度调仓阈值

| 情形 | 阈值 | 操作 |
|------|------|------|
| 个股评分下降 | 下降 ≥ 1.5 分 或 评分 < 5.5 | 考虑卖出 |
| 个股评分进入前列 | 排名进入主板前 30 | 考虑买入 |
| 个股仓位偏离 | 偏离目标仓位 > 25% | 再平衡 |
| 行业评分下降 | 下降 ≥ 1.5 分 或 评分 < 5.0 | 减仓该行业至 5% 以下 |
| 宏观仓位变化 | 变化 > 8% | 调整总仓位 |
| 高股息底仓股息率下降 | 股息率 < 2.5% | 替换为更高股息标的 |

### 6.2 临时调仓（事件驱动）

#### 触发条件与响应矩阵

| 触发类型 | 具体条件 | 响应时间 | 操作 | 看板标识 |
|----------|---------|---------|------|---------|
| **基本面恶化** | 持仓股单季度净利润同比下滑 > 35% 或环比下滑 > 45% | 财报发布当日 | 评分降至 3 分，次日开盘卖出 70% | 🔴 |
| **基本面恶化** | 持仓股出现重大负面（监管立案、核心产品下架、重大诉讼败诉） | 公告当日 | 立即标记，24 小时内决定，默认卖出 | 🔴 |
| **基本面恶化** | 审计意见变为"非标"或"保留意见" | 年报发布当日 | 次日开盘清仓 | 🔴 |
| **估值过高** | 持仓股 PE-TTM 升至历史 88% 分位以上 | 每日监控 | 减仓 50% | 🟠 |
| **估值过高** | 持仓股 15 个交易日内涨幅 > 40% 且无基本面催化 | 每日监控 | 减仓 40%，启动移动止盈 | 🟠 |
| **技术面破位** | 股价跌破 60 日均线且连续 3 日收于下方 | 每日监控 | 减仓 50% | 🟠 |
| **技术面破位** | 股价跌破 120 日均线（半年线）且放量 | 每日监控 | 清仓 | 🔴 |
| **宏观转向** | 宏观综合评分连续 2 周下降且累计降 ≥ 2 分 | 每周五 | 按新评分降仓位 | 🟡 |
| **宏观转向** | 触发"熊市确认"或"顶部区域"标记 | 即时 | 按标记规则执行 | 🔴 |
| **行业利空** | 持仓行业出现重大政策利空（如集采扩围、环保限产） | 政策发布当日 | 该行业股票 3 日内减仓至 5% | 🔴 |
| **行业轮动** | 新行业评分超过当前持仓行业 2 分以上 | 每周五 | 1 周内切换仓位 | 🟡 |
| **机会出现** | 目标股评分 ≥ 8.0 且因大盘暴跌超跌 > 12% | 每日监控 | 启动临时建仓，单只 ≤ 5%，总临时建仓 ≤ 15% | 🟢 |
| **机会出现** | 行业迎来重大政策催化 | 政策发布当日 | 3 日内加仓该行业至 15% | 🟢 |
| **涨停板处理** | 持仓股连续 2 日涨停 | 第 2 个涨停日 | 减仓 30%（获利了结部分） | 🟡 |
| **跌停板处理** | 持仓股跌停 | 跌停当日 | 挂卖单排队，次日继续挂；若连续 2 日跌停，第 3 日开盘挂跌停价清仓 | 🔴 |
| **停牌处理** | 持仓股停牌 > 5 个交易日 | 第 5 日 | 复牌后根据停牌原因处理；若重组失败，开盘卖出 | 🟡 |

### 6.3 调仓执行纪律

```
1. 先卖后买：确保卖出资金释放后再买入，避免资金不足
2. 分批执行：单票调仓幅度 > 4% 时分 2 ~ 3 日完成
3. 价格限制：
   - 买入价不高于前收盘价 2%（防止追高）
   - 卖出价不低于前收盘价 2%（防止恐慌抛售）
   - 若触发涨跌停，按涨跌停规则处理
4. 时间规避：
   - 财报发布前 3 日不新建仓
   - 重大政策发布当日不操作
   - 长假前 3 日不新建仓
5. 成交确认：每笔交易需确认成交，未成交订单需在当日复盘时标记并制定次日方案
```

## 7. 风控体系（低回撤核心）

### 7.1 组合层面风控红线

| 红线 | 阈值 | 触发后操作 | 优先级 |
|------|------|-----------|--------|
| 单只股票仓位上限 | 12% | 禁止买入，超出部分强制减仓 | 最高 |
| 单个行业仓位上限 | 25% | 禁止买入该行业，超出部分强制减仓 | 最高 |
| 总股票仓位下限 | 0%（极端）/ 25%（正常） | 仅"熊市确认"时可降至 0% | 高 |
| 总股票仓位上限 | 90% | 保留最低 10% 现金 | 高 |
| 组合最大回撤 | 10% | 触发强制减仓机制 | 最高 |
| 组合年化波动率 | 18% | 超过则降低个股集中度，增加高股息底仓 | 中 |
| 组合 Beta | 1.0 | 超过则增配低 Beta 股票 | 中 |
| 高股息底仓最低比例 | 股票仓位的 30% | 熊市时必须满足 | 高 |

### 7.2 个股层面风控

| 风控类型 | 触发条件 | 操作 | 执行时间 |
|----------|---------|------|---------|
| **硬止损** | 买入后亏损 ≥ 8% | 无条件清仓 | 触发次日开盘 |
| **移动止盈** | 盈利后从最高点回撤 ≥ 6% | 减仓 50% 或清仓 | 触发当日收盘后决策，次日执行 |
| **时间止损** | 买入后 45 个交易日涨幅 < 3% 且评分下降 | 清仓 | 第 45 日收盘后决策 |
| **基本面止损** | 季报不及预期（利润增速 < 预期 40%） | 次日开盘卖出 70% | 财报发布次日 |
| **流动性止损** | 连续 5 日日均成交 < 3000 万 | 5 日内清仓 | 发现即启动 |
| **跌停止损** | 连续 2 日跌停 | 第 3 日开盘挂跌停价清仓 | 第 3 日开盘 |
| **年线止损** | 股价跌破年线且连续 10 日收于下方 | 清仓 | 第 10 日收盘后决策 |

### 7.3 回撤控制机制（核心中的核心）

每日收盘后计算组合净值回撤：

```
当前回撤 = (历史最高净值 - 当前净值) / 历史最高净值 × 100%

历史最高净值 = max(成立以来每日净值)
```

| 回撤幅度 | 风险等级 | 看板标识 | 应对措施 | 恢复条件 |
|----------|---------|---------|---------|---------|
| < 4% | 正常 | 🟢 | 无需操作，正常监控 | - |
| 4% ~ 6% | 关注 | 🟡 | 暂停开新仓；评分 < 6 的股票减仓 30%；检查宏观环境 | 回撤回到 < 3% |
| 6% ~ 8% | 预警 | 🟠 | 股票仓位降至 55%；只保留评分最高的 4 ~ 6 只；高股息底仓比例提升至股票仓位 50%；增配现金 | 回撤回到 < 4% 且宏观评分 ≥ 5 |
| 8% ~ 10% | 紧急 | 🔴 | 股票仓位降至 35%；只保留评分 ≥ 7 的股票；其余清仓；全面暂停；启动深度复盘 | 回撤回到 < 5% 且宏观评分 ≥ 6 连续 2 周 |
| > 10% | 危机 | ⛔ | 股票仓位降至 15% 以下或清仓；只保留 1 ~ 2 只高股息蓝筹；现金 > 85%；进入"冬眠模式" | 宏观评分连续 1 个月 ≥ 6 且回撤回到 < 6% |

**冬眠模式规则**：
- 股票仓位 ≤ 15%
- 只持有股息率 > 4% 的蓝筹（银行、电力、煤炭）
- 暂停所有新建仓和调仓（除止损外）
- 每周生成一次市场观察报告，不生成操作建议
- 退出条件：宏观评分连续 4 周 ≥ 6 分

### 7.4 尾部风险应对

| 风险类型 | 识别信号 | 应对措施 | 恢复条件 |
|----------|---------|---------|---------|
| **系统性暴跌** | 沪深300 单日跌幅 > 4% 或连续 3 日跌幅 > 7% | 次日开盘减仓 25%；若继续跌，每日再减 15% 直至仓位 ≤ 30% | 沪深300 连续 5 日站稳 20 日均线 |
| **流动性危机** | 两市成交额 < 4500 亿且持续 3 日 | 减仓至 25%，保留高股息蓝筹 | 成交额连续 5 日 > 6000 亿 |
| **黑天鹅事件** | 突发战争、金融危机、重大疫情、地缘冲突升级 | 立即减仓至 20% 以下；持有现金 + 黄金 ETF（若有） | 事件明朗 + 市场企稳 2 周 |
| **风格极致分化** | 某一风格连续暴涨而其他暴跌（如只涨中字头） | 检查组合风格暴露；若偏离市场风格，适度跟随但不追高 | 风格趋于均衡 |
| **汇率剧烈波动** | 离岸人民币汇率单日波动 > 1% 或破 7.3 | 减仓 15%，规避外资流出风险 | 汇率稳定 1 周 |

```

  - 当前持仓 vs 新 Top 100 对比
  - 生成完整调仓方案（卖/买/调/持有）
  - 计算调仓后组合预期指标（预期 PE、ROE、股息率、Beta、波动率）
  - 进行压力测试（假设大盘跌 10%，组合跌多少？）

Step 4：用户审阅与确认（T+1 日开盘前）
  - 看板展示完整调仓方案（含理由、预期、风险）
  - 用户操作选项：
    ├─ 【完全跟盘】一键确认执行

    ├─ 【跳过某只】标记不买入/不卖出某只股票
    ├─ 【自定义】用户自行输入操作
    └─ 【全部跳过】本月不调仓

Step 5：执行调仓（T+1 ~ T+5）
  - 按"先卖后买、分批执行"原则操作
  - 每日更新执行进度
  - 未成交订单标记并制定后续方案

Step 6：月度绩效报告（T+5 日后）
  - 上月组合表现完整归因（Brinson 模型）
  - 各决策层级贡献度（宏观择时、行业配置、个股选择）
  - 风控事件回顾与评估
  - 交易成本分析
  - 用户跟盘行为分析（若用户调整了 AI 建议）
  - 下月展望与策略要点
```

```

  - 策略改进建议

Step 4：年度策略展望（Q4 额外）
  - 下一年度宏观经济预判
  - 下一年度重点行业方向
  - 策略风格定位（价值/成长/均衡）
```

## 10. 绩效归因与复盘

### 10.1 Brinson 归因模型（A股主板适配版）

```
组合超额收益 = 宏观择时效应 + 行业配置效应 + 个股选择效应 + 交互效应 + 高股息效应

【宏观择时效应】
  = (组合平均股票仓位 - 100%) × 基准收益
  （反映仓位择时的贡献，现金比例的影响）

【行业配置效应】
  = Σ[(组合行业权重 - 基准行业权重) × 基准行业收益]
  （反映超配/低配行业的贡献）

【个股选择效应】
  = Σ[组合行业权重 × (组合行业收益 - 基准行业收益)]
  （反映行业内选股的贡献）

【高股息效应】
  = 高股息底仓收益 - 同等仓位基准收益
  （反映防御底仓的独立贡献）

【交互效应】
  = 总超额 - 择时 - 行业 - 个股 - 股息
  （剩余部分，含交易成本、现金拖累等）

基准：沪深300 指数
```

### 11.5 A股主板特殊事项处理

#### 涨跌停处理

| 情形 | 处理规则 |
|------|---------|
| 持仓股涨停（+10%） | 正常持有，不操作；连续 2 日涨停则减仓 30% |
| 持仓股跌停（-10%） | 立即挂卖单排队；次日继续挂；连续 2 日跌停第 3 日开盘挂跌停价清仓 |
| 买入目标涨停 | 不追，取消当日买入计划，次日视情况再议 |
| ST 股跌停（-5%） | 同上，但优先级更高，因 ST 流动性更差 |

#### 停牌处理

| 停牌天数 | 处理规则 |
|---------|---------|
| 1 ~ 5 日 | 正常持有，不操作 |
| 5 ~ 10 日 | 第 5 日复盘时标记"长期停牌"，调整仓位计算（按最后交易价） |
| > 10 日 | 按最后交易价的 90% 计算虚拟净值；复牌后根据原因处理 |
| 复牌后 | 重组成功 → 持有观察 3 日；重组失败/利空 → 开盘卖出 |

#### 分红/送股/配股处理

| 事件 | 处理规则 |
|------|---------|
| 分红除权 | 成本价自动调整；现金分红计入现金仓位 |
| 送股/转增 | 持股数量自动调整；成本价自动调整 |
| 配股 | 看板提醒用户；若 AI 看好则建议参与，不看好则卖出 |
| 定增 | 看板标记；一般不影响持仓决策 |

#### 新股/次新股排除

```
上市不满 2 年的股票（即上市日期 > 当前日期 - 730 天）不纳入选股池。

原因：
  - 主板新股上市后常有炒作，估值失真
  - 财务数据历史短，难以评估稳定性
  - 机构覆盖少，信息不对称大
  - 流动性不稳定
```
